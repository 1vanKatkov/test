# Telegram AI Боты

Коллекция Telegram ботов, которые используют OpenRouter API для генерации ответов.

## Боты

### 1. bot_number.py - Бот Нумеролог 🔮
Бот для нумерологических гаданий с интерактивным меню.

**Возможности:**
- 💕 **Гадание на идеального партнера** - узнайте, каким должен быть ваш идеальный партнер по дате рождения
- 🔮 **Код судьбы** - рассчитайте ваше число судьбы и узнайте свое предназначение
- ✨ **Задать вопрос вселенной** - получите мудрый ответ на любой ваш вопрос

### 2. bot_sonnik.py - Простой AI Бот (если создан)
Простой бот, который отвечает на любые сообщения через AI.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

1. Получите токен бота от [@BotFather](https://t.me/BotFather) в Telegram:
   - Отправьте команду `/newbot`
   - Следуйте инструкциям
   - Скопируйте полученный токен

2. Откройте нужный файл бота и замените `YOUR_TELEGRAM_BOT_TOKEN_HERE` на ваш токен:
```python
TELEGRAM_BOT_TOKEN = "ваш_токен_от_BotFather"
```

## Запуск

Для бота нумеролога:
```bash
python bot_number.py
```

Для простого AI бота (если создан):
```bash
python bot_sonnik.py
```

## Использование

### Bot_number (Нумеролог):
1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Выберите одну из трех кнопок:
   - Гадание на идеального партнера (нужно ввести дату рождения)
   - Код судьбы (нужно ввести дату рождения)
   - Задать вопрос вселенной (введите ваш вопрос)
4. Бот отправит "думаю..." и затем заменит это сообщение на ответ

### Bot_sonnik (если создан):
1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Напишите любое сообщение боту
4. Бот отправит "думаю..." и затем заменит это сообщение на ответ от AI

## Примечание

API ключ OpenRouter уже настроен в коде. Если вы хотите использовать другую модель или другой API ключ, измените соответствующие переменные в файлах ботов:
- `OPENROUTER_API_KEY` - ваш API ключ
- `MODEL` - модель для использования

## YooKassa (тестовые платежи)

Бот создаёт платежи в тестовом режиме, после оплаты пользователь нажимает кнопку **«Я оплатил»**, чтобы бот проверил статус и пополнил баланс.

1. Создайте файл `.env` на основе шаблона `env.example`, чтобы хранить пароль от YooKassa:

   ```cmd
   copy env.example .env
   ```

   или вручную создайте `.env` и заполните переменные:

   ```cmd
   set YOOKASSA_SHOP_ID=ваш_test_shop_id
   set YOOKASSA_SECRET_KEY=ваш_test_secret_key
   set YOOKASSA_ACCOUNT_ID=ваш_test_account_id
   set YOOKASSA_RETURN_URL=https://t.me/Ваш_бот
   set YOOKASSA_TEST_MODE=1
   ```

2. `YOOKASSA_RETURN_URL` должен быть телеграм-ссылкой, которую пользователь увидит после оплаты (например, `https://t.me/ваш_бот`).

3. Доступные тарифы:

   - Стандартные подписки: `150 искр (1 месяц)` — 149₽, `450 искр (3 месяца)` — 399₽, `900 искр (6 месяцев)` — 749₽.
   - После окончания подписки всё несгоревшее количество искр удаляется, включая любые докупки.
   - Пока подписка активна, доступны только доппокупки `50 искр — 100₽` и `100 искр — 200₽` — они также сгорают в конце периода.

4. Убедитесь, что зависимости обновлены (`pip install -r requirements.txt`), чтобы установить библиотеки `yookassa` и `python-dotenv`.

5. После запуска бота команда в меню создания платежа покажет ссылку для перехода на YooKassa. Как только пользователь завершит оплату, нажмите кнопку **«Я оплатил»** — бот сверит статус и добавит Астральные Искры.
